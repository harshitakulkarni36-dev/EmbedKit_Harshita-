# EmbedKit_Harshiita

Name: Harshiita

## Module

### ringbuf.c

Implementation of an 8-byte circular buffer (ring buffer) for uint8_t data.

## Build

Compile:

gcc -Wall -std=c99 ringbuf.c -o ringbuf

Run:

./ringbuf

## Description

ringbuf.c:
Implements a fixed-size circular buffer with support for initialization, write, read, full/empty checks, and count tracking.
